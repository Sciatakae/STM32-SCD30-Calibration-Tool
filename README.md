# STM32-SCD30-Calibration-Tool
This tool can be used to calibrate an SCD30 sensor using an STM32 microcontroller. On boot, a menu in the terminal appears and you can start/stop streaming sensor data with ASC calibration enabled/disabled at your discression; currently no FRC feature.

The SCD30 driver is taken from https://github.com/Sensirion/embedded-i2c-scd30
